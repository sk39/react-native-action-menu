# react-lib-boilerplate

Boilerplate for react native expo library.

## Install

```bash
npm install --save <your-package>
```

## Usage

```jsx
import React, { Component } from 'react'

import YourComponent from <your-package>

class Example extends Component {
  render () {
    return (
      <YourComponent />
    )
  }
}
```
